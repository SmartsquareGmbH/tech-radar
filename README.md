# Smartsquare Tech Radar

**Live version: https://smartsquaregmbh.github.io/tech-radar/**

A visualization of Smartsquare's technology landscape, based on
the [Zalando Tech Radar](https://opensource.zalando.com/tech-radar/) and inspired
by [ThoughtWorks](https://www.thoughtworks.com/radar).

## What is the Tech Radar?

The Tech Radar is a tool to inspire and support engineers at Smartsquare in picking the best technologies for new
projects. Technologies are assessed and placed into one of four rings:

- **ADOPT** — High-confidence technologies recommended for wide use in production.
- **TRIAL** — Technologies proven in project work; slightly more risky but with shared team experience.
- **ASSESS** — Promising technologies worth researching and prototyping; higher risk, often newer.
- **HOLD** — Not recommended for new projects; existing usage can continue.
